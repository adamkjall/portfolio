Booktropolis

Booktropolis is my first React project. It feches data from New York Times Books API and shows this weeks best sellers as well as information about each book on the list.

You can also view previous weeks top lists aswell as search for any book and the search will query Google Books API and view the result in the same fashion.

In this project I learned the basics of React and how to fetch data from API's. I also learned the basics of how to use redux to manage the state of the appliation.

Built with:

  • HTML5

  • CSS3

  • JavaScript

  • React

  • Redux

SimplePark

SimplePark is an android/android car app, it tries to help with finding a good place to park in Gothenburg by viewing the parking lots on a map with information about prices, time and spots left.

I developed this app with a team of students taking the Software Engineering Project course at Chalmers. The task was to create an android app which should be able to run on Volvo cars android OS and make the life of Volvo owners more enjoyable while commuting.

On this project i learned about agile development and teamwork, we worked according to the SCRUM methodology and documented every step on github.

Built with:

  • Java

  • Android Studio

Bait Boat

BaitBoat is an app I developed for a company that wanted to test if they could control their bait boat prototype with a smartphone.

This was a very challenging project with alot of technical parts that I had never done before, e.g. socket communication between server and client.

Built with:

  • Java

  • Python

  • Android Studio
